# Machine Learning – MPG Prediction

This project uses the Auto MPG dataset to train a Linear Regression model that predicts
mileage (MPG) based on car features.

## Features:
- cylinders
- displacement
- horsepower
- weight
- acceleration
- model_year
- origin

## Model:
Linear Regression (sklearn)
